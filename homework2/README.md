# Завдання №2. SQL для Data Engineering

## Послідовність виконання завдання

1. Встановити Docker Compose на локальний компʼютер. Спосіб встановлення може відрізнятися в залежності від того, яка ОС використовується.
2. Встановити Демо базу даних

  - Склонувати репозиторій Демо бази Pagila.
  - Запустити Docker Compose, який підніме Демо базу та заповнить данними. Це можна зробити наступними способами:

        ➡️ зайти у кореневу директорію репозиторію Демо бази та виконати команду docker-compose up

        ➡️ відкрити файл docker-compose.yml в Pycharm та запустити Docker Compose за допомогою Pycharm

3. Підʼєднатись до Демо бази використовуючи або Pycharm Profesional або DBeaver. За бажанням можна використати будь-який інший звичний клієнт роботи з базами даних. Креди для підʼєднання до демо бази можна знайти в файлі docker-compose.yml (user, password, port)

4. Написати SQL код до всіх задач у файл шаблону.
